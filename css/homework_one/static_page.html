作业一：静态网站制作

板块一：基本介绍

  当用户向网站访问一个网页，网站会把这个网页文件传给用户的客户机，然后客户机的浏览器就打开并展示信息内容。而传到客户机的网页文件并没有自身打开，也不需要自身打开。 我们看话剧的时候看到的是演员照着剧本演出的结果，观众是不用看剧本的。
  网页文件就是剧本或叫做脚本，隐藏在后台，其中包含演出内容和如何进行演出的指令。而浏览器就是演员，它在前台按照网页文件这个“剧本或者脚本”事先写好的内容和规定进行表演、展示。用户看到的是浏览器(遵照网页文件内写好的规定)进行“演出”的结果。

  HTML（HyperText Markup Language）是超文本标记语言的缩写，通过HTML，我们利用标记来标示内容提供结构，并把匹配标记以及它们所包围的内容称为元素，然后浏览器会按照内置的默认规则显示各个元素。

  • HTML 与 CSS 是我们用来创建网页的语言。
  • 元素由三部分组成：一个开始标记、内容和一个结束标记，但有些元素有例外（如<br>、<img>）。
  • 开始标记可以有属性。
  • 结束标记名前面有一个“/”，以确定这是一个结束标记。
  • 所有页面都要有一个<HTML>元素，其中要有一个<head>元素和<body>元素，<head>元素存放网页的信息，<body>元素里的内容就是你将在浏览器里看到的东西。
  • 大多数空白符（制表符、回车、空格）都会被浏览器忽略，不过可以利用空白符让你的 HTML（对人）更具有可读性。
  • 可以在<style>元素中写 CSS 规则，来指定元素的特性，且<style>元素应放在<head>里。
  • HTML 标签对大小写不敏感，但为了可读性，建议书写一致。
  • “<!-”与 "->"之间放置的所有内容都会被浏览器忽略！单引号“ ' ”可以注释一行。
  当然，我们完全可以不拘泥于默认设置，可以通过 CSS（Cascading Style Sheets）层叠样式表 来增加自己的样式和格式化规则，来确定字体、颜色、大小和页面的很多其它特征（元素中属性的用处也是这样），控制 HTML的表现。
  下面开始介绍一些超文本标记。

板块二：超文本标记

引言

  大部分标记都是成对出现的，即一个开始标记和一个对应的结束标记。每一对标记就像是一对特殊括号，它们会把要展示的文章的一部分括起来，规定这部分内容要以某种位置、大小、颜色、字体进行突出展示，以便区别文章的其他部分，使得网页的版面丰富多彩。
  只有少量的标记是单个的，而不是成对的。

基本框架

<HTML>
<head>
<title>标题</title>
</head>
<body>
</body>
</HTML>

  1. 网页开始标记：<HTML>，网页结束标记：</HTML>
  2. 头部开始标记：<head>网页结束标记：</head>
  3. 浏览器标题栏标记：<title> 和 </title>
  4. 网页窗体开始标记：<body>，网页窗体结束标记：</body>
    （网页中除了头部之外的所有内容都被他们括住，并在浏览器最主要的部分—窗体上显示）

样式与排版

  1. 窗体背景颜色

    可在窗体开始标记中加入"bgcolor=某种颜色表示"，如<head bgcolor="某种颜色表示">

    标记中添加的"bgcolor=某种颜色表示"称为属性，通过属性，我们可以表达一些特殊的需求。
    属性标记一般都写在开始标记中，任何一个开始标记中可写入多个属性标记，且属性的顺序可任意。

    再谈谈Web颜色的表示：Web颜色是按照构成颜色的红、绿、蓝三个分量所占数量来指定的，每种颜色会分别指定一个从0到100%的数值，然后把它们混合起来得到最终的颜色，如果把均为100%的红、绿、蓝混合在一起，就会得到白色！
    指定颜色的方法：

    • 按名指定

      如直接使用red、black、blue等等，但这种方法所能指定的颜色很有限。

    • 用红、绿、蓝指定

      如白色用 rgb(100%,100%,100%) 表示，橙色用 rgb(80%,40%,0%) 表示。
      还可以将红、绿、蓝指定为0到255之间的一个数值，如橙色可用 rgb(204,102,0) 表示。

    • 使用十六进制码指定
      以“#”开头，其中每组两位数字分别代表颜色的红、绿、蓝分量，每位数都是一个十六进制数，橙色可用 #cc6600 表示，可将每组数转化为一个0到255的数，转化方法如下：

    段落标记为<p>和</p>，它是一个块元素，块元素与上下文之间都有空行。

  5. 空格

    一个汉字占两个空格大小，所以使用四个空格就可以达到首行缩进两个汉字的效果。

    空格标记有如下几种表示：

    • &nbsp;

      不换行空格，全称是 No-Break Space，它是最常见和我们使用最多的空格，是按下space键产生的空格，故要使用四个该标记才能缩进两个汉字。

    • &ensp;

      它叫“半角空格”，全称是 En Space，en是字体排印学的计量单位，为em宽度的一半。根据定义，它等同于字体度的一半（如16px字体中就是8px）。此空格有个相当稳健的特性，就是其占据的宽度正好是1/2个中文宽度，而且基本上不受字体影响。

    • &emsp;

      它叫“全角空格”，全称是 Em Space，em是字体排印学的计量单位，相当于当前指定的点数。例如，1 em在16px的字体中就是16px。此空格也有个相当稳健的特性，就是其占据的宽度正好是1个中文宽度，而且基本上不受字体影响。

  6. 回车

    回车标记为<br>，每使用一个回车标记便会换一行，该标记是一个空（void）元素，即该元素没有内容，当需要使用 void 元素时，只需使用一个开始标记，这是一种方便的简写形式，可以减少 HTML 中的标记数量。

  8. 超链接标记

    标记为<a>和</a>，在网页上一段文字或一个图片，被这一对儿特殊的括号括起来之后，就成为超级连接，用户点击这个超级连接之后，就会从当前的网页转向（打开)另外一个(目标)网页。

    为了用户点击超级连接后，用户的浏览器能够转向目标网页，就必须在开始标记中写出目标网页的网址，属性标记 href="目标地址",就是做这个用途的。
    href（hypertextreference）的含义是超文本引用，目标地址可以使用相对路径或绝对路径（HTML 中的路径都要用“ / ”）。

    在浏览器输入的 Web 地址称为 URL，或统一资源定位符（Uniform Resource Locators），它是一个全局地址，可用来定位 Web 上的任意资源，包括 HTML 页面、音频、视频和其它很多形式的 Web 内容。
    除了指定资源的位置，URL 还可以指定用来获取资源的协议：HTTP（HyperText Transfer Protocol）也称为超文本传输协议。

    title 属性：用来提供链接信息。
    target 属性：可以控制浏览器打开一个新的窗口（或新标签页）来显示链接到的网页。设置为 target="_blank"。

    使用<a href="https://vpn.muc.edu.cn/portal/#!/login" title="前往民大VPN登陆界面" target="_blank" >前往民大VPN登陆界面</a>，效果为：前往民大VPN登陆界面

表单

表单就是像人们申请某项职位，填写自己的情况信息的申请单，在网页上就是多个输入框，用户在框里输入信息之后，按提交按钮向网站发送。正是有了表单，用户可以在网页上向网站提出特定的要求信息，向网站发送后，网站就会接收这些用户信息，分析用户的要求，按照用户的要求在网站中找到用户需要的信息，做成回复网页传给客户。

  1. 开始、结束标记

    标记为<form>和</form>，所有的表单输入框和结束按钮都是该元素的内容。

  2. 单行文本输入框标记

    标记为<input type=text name="变量名">，这种输入框最常用，可用于输入姓名、电话号码等等。

    注意：

    • type属性

      type="某种输入框类型"
      输入框中的 type 属性指定了输入量的类型，type="text" 表示这是一个单行文本输入框。

    • name属性

      name="变量名"
      每个输入框都要设置一个变量，用户输入的信息会保存在这个变量当中，最后向网站发送,没有变量，用户输入的信息就无法保存。变量名是网页制作人自己起的,最好与它的用途保持一致，使人一看就明白。

    • value属性

      value="预先放在输入框的值（提示）"
      这个属性可以没有，如果有的话，输入框当中预先放好了值。用户如果对预设值不满意当然可以对这个值进行修改。

    每种输入框，包括接下来在下面要讲的输入框，在它们的上、下、左、右之一，应该用普通文字写出该输入框用途的提醒文字，以便提示用户输入正确的信息。

    对于代码：

    <form align="center">
    姓名：<input type="text" name="name1"><br>
    电话：<input type="text" name="tele" value="888-8888">
    </form>

    显示为
    姓名：
    电话：

  3. 口令输入框

    具有这两个属性：type="password" maxlength="输入框接受的字符最大数"

    用户在口令框里输入字符时，不显示真实的字符而用圆点代替。

    对于代码：

    <form align="center">
    密码：<input type="password" name="mima" maxlength="6">
    </form>

    显示为
    密码：

  4. 多行文本输入框

    标记为<textarea>和</textarea>，这是一个成对儿的标记，在开始标记中要加入表示行数和列数的属性标记 cols="列数" 和 rows="行数"。

    对于代码：

    <form align="center">
    <p>请输入你的答案：（最多100字）</p>
    <textarea cols="36" rows="6" name="text" style="resize:none" maxlength="100"></textarea>
    </form>

    显示为

    请输入你的答案：（最多100字）

  5. 单选输入框

    类型属性为 type="radio"，可以根据需要设置多个<input>选项，用户将来只能选择一项。每个选项当中还有 value="某值" 的属性。
    同时每个选项，还有变量名。值得注意的是，所有选项的变量名应该是同一个变量。将来用户选择某一项时，该项的 value 值就会进入到这个同一变量中保存并向网站发送。当网站接收这个变量，检查其中的值，就知道用户在多项中选择了哪一项。
    当某一项中出现了checked的属性，那么该项就会被预先选中，但是如果用户不满意，可以再点选消除。

    对于代码：

    <form align="center">
    <p>请选择你最喜欢的运动：</p>
    <input type="radio" name="sports" value="跑步" checked>跑步<br>
    <input type="radio" name="sports" value="游泳">游泳<br>
    <input type="radio" name="sports" value="登山">登山
    </form>

    显示为

    请选择你最喜欢的运动：

    跑步
    游泳
    登山

  6. 多选输入框

    类型属性为 type="checkbox"，可以根据需要设置多个<input>选项，用户将来能选择其中的多项。每个选项当中还有 value="某值" 的属性。
    同时每个选项，还有变量名。值得注意的是，所有选项的变量名应该不是同一个变量，也就是每一个选项的变量都是一个单独的不同变量。这一点跟单选输入框是不一样的。将来用户选择某一项时，该项的 value 值就会进入到这个选项的特定变量中保存并向网站发送。当网站接收这些个变量，检查其中有值的变量就知道用户在多项中选择了哪些项。
    当某一些项中出现了checked的属性，那么这些项就会被预先选中，但是如果用户不满意，可以再勾选它们消除。

    对于代码：

    <form align="center">
    <p>请选择你喜欢的节日：（可多选）</p>
    <input type="checkbox" name="festival1" value="元旦节" checked>元旦节<br>
    <input type="checkbox" name="festival2" value="中秋节">中秋节<br>
    <input type="checkbox" name="festival3" value="端午节">端午节<br>
    <input type="checkbox" name="festival4" value="圣诞节">圣诞节<br>
    <input type="checkbox" name="festival5" value="劳动节">劳动节
    </form>

    显示为

    请选择你喜欢的节日：（可多选）

    元旦节
    中秋节
    端午节
    圣诞节
    劳动节

  7. 下拉菜单标记

    标记为<select name="变量">和</select>，在这一对标记之间，可以根据需要设定多个选项<option>,每个选项的右边应该有提示性文字。用户只能选择一个选项。下拉菜单儿在网页中平时只有一行，点击其上的下箭头，下拉菜单会展开各个选项，供用户点选其中之一。

    对于代码：

    <form align="center">
    <p>你最喜欢哪个城市？</p>
    <select name="city">
    <option>北京
    <option>上海
    <option>广东
    <option>江西
    </select>
    </form>

    显示为

    你最喜欢哪个城市？

  8. 滚动菜单标记

    添加属性 multiple size="露出的行数"，在这一对标记之间，可以根据需要设定多个选项<option>，每个选项的右边应该有提示性文字。在开始标记中出现 multiple 属性，意味着将来用户可以进行多选（按住ctrl进行多选），如果没有 multiple 属性，用户只能选择一个选项。
    如果开始标记中的最后一个属性写成size=5，滚动菜单儿在网页中只漏出5行，点击其上的滚动条会看到其它的选项。

    对于代码：

    <form align="center">
    <p>你到过哪些城市？</p>
    <select name="city" multiple size="3">
    <option>北京
    <option>上海
    <option>广东
    <option>江西
    <option>福建
    <option>哈尔滨
    </select>
    </form>

    显示为

    你到过哪些城市？

  9. 提交标记和重置按钮标记

    标记为：
    <input type="submit" value="提交">
    <input type="reset" value="重置">

    用户点击提交按钮之后，本表单所有输入框的变量，也就是表单变量就立即装着用户输入的各种信息向网站发送。网站接到这些变量，会取出它们保存的用户信息进行分析，然后酌情分别处理最后给用户答复。

    对于代码：

    <form align="center">
    <input type="submit" value="提交">&nbsp;&nbsp;<input type="reset" value="重置">
    </form>

    显示为
